HALOLINK v0.1.6 — START HERE

1. Mac: double-click mac-bridge/run_bridge.command
2. Chrome: chrome://extensions -> Developer mode -> Load unpacked -> choose chrome-extension
3. Android: install apk/HaloLink-v0.1.6-debug.apk and open HaloLink on the same Wi-Fi
4. Confirm http://MAC_IP:PORT/health reports product=HaloLink and phoneClients=1
5. Test animations: double-click mac-bridge/demo_states.command

HaloLink prefers port 8766 and falls back through 8775. It does not use the
Camera Activity Notifier service on 127.0.0.1:8765.

The project uses Wi-Fi/LAN auto-discovery. Bluetooth is not required.
